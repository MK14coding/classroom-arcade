Movie Pattern Quest

Unlock movie tickets by solving pattern challenges.

How to Play

There are 60 movie tickets to unlock across the whole game.

Each game type has 10 different movie questions. For example, Missing Number - Easy, Missing Number - Medium and Missing Number - Hard have 30 movies to unlock together.

Answer correctly to unlock that movie ticket. Re-play a game type to unlock any movies you missed.

Movie names are text references only. No movie images or logos are used.
